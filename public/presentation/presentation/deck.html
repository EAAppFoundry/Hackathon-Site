Goals

  • Learn something new

    Get out of our comfort zone and use new tools and technologies

  • Recharge our batteries

    Break the monotony of the day in/day out of current projects

  • Build some apps

    Go from conception -> design -> build -> deploy in 1 week

What Did We Build?

  • Loggly

    A way to capture and view logfiles in the cloud

  • Apollo

    A search engine for requirements and code

  • Infinity

    A comprehensive view of linear and non-linear schedules for both Turner and Non-Turner networks

What Technology Did We Use

  • node.js

    node.js platform that allows developers to build web applications (both client and server) in JavaScript.

  • MongoDB

    mongodb is a high-performance, open-source, document-oriented database.

  • Git and Github

    Git is an open-source, distributed version control system. Git makes it easy for large teams to work in a parallall and disconnected manor. Github is a web-based hosting service for software projects using Git. Git hub makes it easy to share source code with the rest of the open-source community.

  • Trello

    Trello is a collaboration tool that organizes projects into boards, making it easy to tell at a glance what tasks are done, what is in flight, and what's coming up next.
